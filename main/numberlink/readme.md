# ナンバーリンク v2.0 (パズルゲーム)

Pyxelを使用したナンバーリンクパズルゲームです。同じ数字を線で繋いでください。

## スクリーンショット


## ブラウザ上での実行
- こちらからプレイできます。[ブラウザで遊ぶ](https://kitao.github.io/pyxel/wasm/launcher/?play=ku-ron.pyxel_minigames.main.numberlink.numberlink_v2_0&gamepad=enabled)


## ゲームのルール
- 同じ数字のペアを線で繋ぎます。
- 線は分岐や交差はできません。
- 全部の数字のペアを線で繋げばクリアです。

## 収録パズル数
- 5x5: 1問
- 6x6: 12問
- 8x8: 105問
- 10x10: 27問
- **合計: 145問**

## 操作方法

### キーボード
| キー | 動作 |
|------|------|
| 矢印キー | カーソルの移動 |
| Enter | メニューでの決定、クリア後にメニューへ戻る |
| Space | 移動モード/描画モードの切り替え |
| R | ゲームのリセット |
| M | メニューに戻る |
| B | BGMのオン/オフ |
| ESC | ゲーム終了 |

### ゲームパッド
| ボタン | 動作 |
|--------|------|
| 十字キー | カーソルの移動 |
| B (右) | 決定、クリア後にメニューへ戻る |
| A (下) | 移動モード/描画モードの切り替え |
| Y (上) | メニューに戻る |
| X (左) | ゲームのリセット / BGMオン/オフ |

## 遊び方
1. メニューで遊びたいパズルを選択
2. ゲーム画面でSpaceキー（またはAボタン）で描画モードに切り替え
3. 矢印キーで移動しながら線を引く
4. 同じ数字同士を全て繋げばクリア！
5. クリアしたパズルはメニューで緑色に表示されます（*マーク付き）

## 制作者の実行環境

- Python 3.11.0
- Pyxel 2.3.6

## インストールと実行

1. Pyxelをインストール
   ```
   pip install pyxel
   ```

2. ゲームを実行
   - srcフォルダから実行する場合
      ```
      cd src
      python main.py
      ```
   - pyxappから実行する場合
      ```
      pyxel play numberlink_v2_0.pyxapp
      ```
